# Screenshots

Add screenshots here to showcase Gilfoyle features. Use Cmd+Shift+4 on macOS to capture a region.

## Needed Screenshots

| Filename | Description | How to capture |
|----------|-------------|----------------|
| `main.png` | Hero shot — Power layout with terminal, sidebar, activity | Already captured |
| `power-layout.png` | Power layout close-up | Already captured |
| `simple-layout.png` | Simple layout — clean terminal-only view | Switch to Simple via Cmd+Shift+P or Command Palette |
| `command-palette.png` | Command palette overlay | Press Cmd+K |
| `onboarding.png` | Onboarding wizard (step 1 or 2) | Settings > Reset Onboarding, restart app |
| `activity-feed.png` | Activity tab with file edits and git ops | Click Activity tab while Claude is working |
| `file-browser.png` | File tree with code preview | Click a file in the sidebar |
| `mcp-tools.png` | Tools tab showing MCP servers | Click Tools tab |
| `plugin-discover.png` | Plugin marketplace / discover panel | Click Discover in sidebar |
| `light-mode.png` | Light theme overview | Toggle via Cmd+K > Switch to Light Mode |
| `usage-tracking.png` | Usage meter in title bar | Visible when usage data is available |
